# RDP Brute Force Detection Lab

## 🎯 Objective
Simulate an RDP brute-force attack against a Windows 10 system and develop detection logic using Sysmon and Splunk.

## 💻 Environment
- Windows 10 VM (RDP enabled, Sysmon installed)
- Kali Linux / Parrot OS (Attacker VM)
- Splunk Free or Event Viewer
- Optional: ELK, Sigma

## 🔍 MITRE ATT&CK Mapping
- T1110: Brute Force
- T1021.001: Remote Services (RDP)

## 🛠️ Outcome
- Full attack execution documented
- Detection methods + event correlation explained
- Sigma/YARA rules generated (optional)
